HiCrew: Hierarchical Reasoning for Long-Form Video Understanding via Question-Aware Multi-Agent Collaboration

Yuehan Zhu, Jingqi Zhao, Jiawen Zhao, Xudong Mao, Baoquan Zhao
Sun Yat-sen University
Accepted at ICME 2026
arXiv Code PDF

Overview of the HiCrew framework. Our approach combines a Hybrid Tree structure for video representation, Question-Aware Captioning for targeted visual description, and a Planning Layer for adaptive multi-agent orchestration.

Abstract

Long-form video understanding remains fundamentally challenged by pervasive spatiotemporal redundancy and intricate narrative dependencies that span extended temporal horizons. While recent structured representations compress visual information effectively, they frequently sacrifice temporal coherence, which is critical for causal reasoning. Meanwhile, existing multi-agent frameworks operate through rigid, pre-defined workflows that fail to adapt their reasoning strategies to question-specific demands. In this paper, we introduce HiCrew, a hierarchical multi-agent framework that addresses these limitations through three core contributions. First, we propose a Hybrid Tree structure that leverages shot boundary detection to preserve temporal topology while performing relevance-guided hierarchical clustering within semantically coherent segments. Second, we develop a Question-Aware Captioning mechanism that synthesizes intent-driven visual prompts to generate precision-oriented semantic descriptions. Third, we integrate a Planning Layer that dynamically orchestrates agent collaboration by adaptively selecting roles and execution paths based on question complexity. Extensive experiments on EgoSchema and NExT-QA validate the effectiveness of our approach, demonstrating strong performance across diverse question types with particularly pronounced gains in temporal and causal reasoning tasks.

Framework Overview

HiCrew comprises two main stages: Video Preprocessing (right panel) constructs the Hybrid Tree representation with question-aware captioning, and Hierarchical Multi-Agent Collaboration (left panel) orchestrates specialized agents through adaptive planning. The Planning Layer analyzes question semantics to determine optimal agent composition and execution workflow, enabling efficient and targeted reasoning.

Main Results

We evaluate HiCrew on two long-form video understanding benchmarks: EgoSchema (over 5,000 questions across 250+ hours of egocentric video) and NExT-QA (5,440 videos with ~52K QA pairs covering Causal, Temporal, and Descriptive reasoning). HiCrew achieves new state-of-the-art performance on both benchmarks with consistent improvements across all question types.

Ablation Study

Systematic ablation on EgoSchema subset validates the contribution of each core component. Replacing the Hybrid Tree with uniform frame sampling results in a dramatic 12.9% drop. Reverting to question-agnostic captions decreases accuracy by 12.4%. Removing adaptive planning yields a 9.6% reduction. These results confirm that each component contributes meaningfully, with their combination creating synergistic effects.

Qualitative Analysis

The qualitative example illustrates HiCrew's reasoning process on a challenging NExT-QA causal question: "Why does the man look up?" The video is segmented into four shots via shot boundary detection. Relevance assessment identifies the critical shot for deep hierarchical expansion, while low-relevance shots remain as leaf nodes. The Planning Layer initiates a bidirectional temporal search workflow where specialized agents collaboratively reason about the causal relationship, correctly identifying the answer: "overlooking playing children" (Option B). In contrast, VideoTree disrupts temporal structure through global clustering and misses critical environmental context, incorrectly inferring the man is "looking at a bird."
